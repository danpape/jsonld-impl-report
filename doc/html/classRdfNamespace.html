Detailed Description

Provides context with a URI to a definition and a prefix that can be used as a reference to the URI.

A RDF Namespace contains a URI and a prefix that can be used as a substitute for the URI, making the data more human readable. The URI can either be used inline in the data or replaced inline with the prefix and then included in a context section.

When comparing two RdfNamespace(), we look to the prefix and use the value to determine equality.
