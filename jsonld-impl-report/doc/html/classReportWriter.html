Detailed Description

A class to provide an interface that we'll use to write out our implementation report.

I wanted a way to make sure that I wasn't tying myself doeen to a particular ostream — either writing to the console, file or both — so I thought it would be best to create a class that could be modified and/or extended in future and as long as it maintained the same interrface then it wouldn't impact other areas of the program.
